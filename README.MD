# Personal Portfolio Website

A responsive personal portfolio website showcasing my skills, experience, and education in software development.

## Features

- Responsive navigation menu with hover effects
- Dark theme design with custom accent colors
- Interactive tabs for Skills, Experience, and Education
- Dynamic content switching using JavaScript
- Custom CSS animations and transitions

## Technologies Used

- HTML5
- CSS3 (Flexbox, Custom Properties)
- JavaScript (DOM Manipulation)
- Custom CSS animations
